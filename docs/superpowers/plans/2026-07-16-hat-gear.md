# Hat Gear Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** เปิดความสามารถเต็มรูปแบบของช่องหมวก (`hat`): สเตตพื้นฐาน DEF/HP/DMG, Option หลัก (dropdown ทุกแรงก์ยกเว้น Root Abyss), Flame พูล Hat (6459), Potential + Bonus Potential พูล Hat (6439), Emblem Unique+, ไม่มี Soul และไม่มี Genesis

**Architecture:** Slot override บน `hat` (ไม่เปิดทั้งหมวด `armor`) · พูลใหม่แยกไฟล์ · ขยาย `applyFlameScales` ให้คืน `GearStatBag` เพื่อรองรับ DEF flame · ฟิลด์ `phyDefBase`/`magDefBase`/`maxHpBase`/`maxDamageBase` บน `GearItem`

**Tech Stack:** TypeScript, React, Vitest, Vite planner app

## Global Constraints

- ขอบเขต: เฉพาะ `hat` — อย่าเปิด flame/pot ให้ gloves/cape อื่นในเฟสนี้
- หมวกไม่มี `Genesis`
- Option หลักโชว์ทุกแรงก์ **ยกเว้น** `Root Abyss`
- Emblem เลือกได้เมื่อ rank ≥ `Unique` เท่านั้น
- Emblem +30% บูสต์เฉพาะ `phyDef`+`magDef` ของชิ้น — ไม่บูสต์ HP / Max DMG base
- Soul: ไม่รองรับ
- ตาราง: Flame Hat = Nexon 6459 · Pot/Bonus Hat = Nexon 6439
- Spec: `docs/superpowers/specs/2026-07-16-hat-gear-design.md`
